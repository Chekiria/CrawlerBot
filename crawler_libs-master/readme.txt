## Договоренности
1) Комментарии и doc_string на русском;
2) Функции в любом случае должны возврашщать однинаковый тип данных;
3) Названия функций типа 'дієслово_іменник', к примеру: get_domain(), generate_queue();
4) Функции, которые возвращают несколько значений - должны возвращать словарь;
5) Не пихать куда попало try...except;
6) Стараться делать функции рабочими на linux.

Для использования нужно:
- Перейти: Система панель управления -> Дополнительные параметры -> Переменные среды
- Создать переменную PYTHONPATH
- Установить переменной PYTHONPATH значение - путь к crawler_libs

После этого можно использовать данные модули с любого места на компютере
from api import ProjectsApi
